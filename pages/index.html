Speech to text for lectures

How to use

This is a live captioning program for use in lectures or conferences. If you are listening to a speech, then captions should automatically appear on the screen. Otherwise, the presenter should press the "Start/Stop" button to commence the captioning.

To view the entire transcript of the current speech, press the "Show all text button".
